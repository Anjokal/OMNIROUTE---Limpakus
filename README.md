# OMNIROUTE - Limpakus
Omniroute-LIMPAKUS is a local Chrome extension created to simplify the manual collection and formatting of session credentials required by OmniRoute Web providers. It detects supported AI platforms, reads only the credential type needed for the selected provider, and copies the result to the clipboard so it can be pasted into OmniRoute.
