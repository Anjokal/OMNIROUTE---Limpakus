Privacy, security and responsible use

This page contains the privacy information and legal conditions for Omniroute‑LIMPAKUS.

Local processing

The extension processes cookies, session tokens, authorization headers and values entered by the user only to copy the credential requested for configuration in OmniRoute.

The extension has no external server, does not sell data and does not transmit copied credentials to the author.

Temporary session data

Some request headers may be kept temporarily in chrome.storage.session. This storage is cleared when the browser session ends and can also be cleared from the extension settings.

Account security

Session credentials may allow temporary access to an account. They must be protected like a password and must never be shared with third parties.

Responsible use

Use this extension only with accounts, devices and systems that belong to you or for which you have explicit authorization.

Unauthorized access, credential theft, surveillance, fraud, bypassing security controls and any unlawful or abusive use are prohibited.
